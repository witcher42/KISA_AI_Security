# 사이킷런(Scikit-learn) vs 텐서플로우(TensorFlow)  
http://blog.naver.com/las1001/221720815533   

- 사이킷런 장점
  - 잘 정의된 탄탄한 학습 알고리즘  
- 사이킷런 단점
  - 딥러닝이나 강화학습을 다루지 않음  
  - GPU를 지원하지 않음  
- 텐서플로우
  - 하드웨어가 좋을때 탁월한 성능  
  - 딥러닝에서 많이 사용하는 다양한 모델과 알고리즘이 들어 있음
  - 결과를 설명하는 데이터 플로우 그래프 DFG를 표시하고 이해하기 좋음
